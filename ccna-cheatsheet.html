Blocking — só recebe BPDUs

Learning — 15 s · constrói tabela MAC
Forwarding — totalmente operacional
Disabled — desativado administrativamente
Papéis de Porta

Designated Port — melhor porta por segmento
Alternate (RSTP) — backup do Root Port
Backup (RSTP) — backup do Designated Port
Eleição da Root Bridge
Menor Bridge Priority (padrão 32768) vence
Empate: menor endereço MAC vira root
Prioridade em incrementos de 4096
Timers (802.1D)

ACL Estendida (100–199, 2000–2699)

ACL Nomeada
ip access-list extended NOMEADA 10 permit tcp 10.0.0.0 0.0.0.255 any eq 22 20 deny ip any any # Deletar linha: no 10
Máscaras Wildcard
0.0.0.0 = host exato · 255.255.255.255 = any
0.0.0.255 = rede /24 · host = /32 · any = 0.0.0.0 255.255.255.255
Regras Importantes
Processado de cima para baixo · primeiro match aplica
Deny implícito ao final de toda ACL
Uma ACL por interface por direção (in/out)
